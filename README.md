# NERDHERD BACKEND SERVICE

## FEATURES

- API
  - [OTP](./api/otp.py)
    - [x] get random 6-digit OTP code

## INSTALLATION

1. `pip install -r requirements.txt`
2. `pip freeze > requirements.txt`
